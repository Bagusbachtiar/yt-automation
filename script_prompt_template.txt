3. GROUNDING RULE: Only include claims explicitly stated in the reference text above. Do not add outside knowledge, even if you are confident it is true. If a fact is not in the reference, omit it or describe the phenomenon qualitatively.
4. FACT-CHECKING RULE: Only include numeric or statistical claims that appear in the reference text. If a number is absent from the reference, describe the trend in words instead (e.g. "a major source of" instead of "responsible for 34% of").
5. CAUSAL PRECISION RULE: Do not attribute an effect to a cause unless the direct causal link is stated in the reference text.

9. End with a short, grounded closing line (no call to buy/subscribe, just a thought or takeaway).
